# Scientific Calculator

This project is a fully functional **Scientific Calculator** built using Python.  
It can perform a wide range of mathematical operations, including:

- Addition  
- Subtraction  
- Multiplication  
- Division  
- Power (x^y)  
- Square Root  
- Sine  
- Cosine  
- Tangent  
- Natural Logarithm (ln)  
- Log Base 10  

---

 Features
- Easy-to-use menu system  
- Trigonometric functions with degree input  
- Error handling for invalid inputs  
- Works on Android (Termux), Windows, Linux, and any Python-supported device
